# Prompts

Prompt templates for Ralph's built-in modes (`plan.md`, `build.md`,
`plan-process.md`, `build-process.md`, `align-specs.md`, `sandbox-*.md`)
and the shipped ad-hoc prompts (`adhoc-*.md`) for use with `ralph prompt`.

For end-user guidance on writing custom ad-hoc prompts:

```
ralph help prompt
```

For the design intent and structural contract governing the shipped
ad-hoc prompts, see [../specs/adhoc-prompts.md](../specs/adhoc-prompts.md).
